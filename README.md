# 🛡️ AI Scam Detection System

## 📌 Overview

The AI Scam Detection System is designed to detect fraudulent or phishing messages in real-time.  
It uses text preprocessing, feature extraction (TF-IDF), and a trained ML model to classify suspicious content.

This system can be integrated into:
- Banking platforms
- Email filtering systems
- Messaging applications
- Fraud detection pipelines

---

## 🚀 Features

- ✅ Real-time scam detection
- ✅ Machine Learning-based classification
- ✅ Confidence score output
- ✅ REST API support
- ✅ Modular project structure
- ✅ Easy deployment

---

## 🧠 How It Works

1. User inputs text
2. Text preprocessing (cleaning, stopword removal)
3. Feature extraction using TF-IDF
4. ML model prediction
5. Output classification with confidence score
